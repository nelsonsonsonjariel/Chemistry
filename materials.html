Procedure


1. First of all we need to prepare all our materials and then we add a pour small amount of cabbage juice into each cup or test.

2. Then we carefully add a small amount of a test tube substance to the juice.

3. We gently stir and observe the color change.

4. We record the color and decide if the substance is acid or if it is a base or in another case a neutral.

5. We repeat the same process with the other substances.

6. Finally you need to clean out all the test tubes you use.
